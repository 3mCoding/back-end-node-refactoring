형식
if(조건식1) {
	// 조건식1이 참일 경우 실행
	실행문1;
	실행문2;
}
else if(조건식2) {
	// 조건식2가 참일 경우 실행
	실행문1;
	실행문2;
}
else {
	// 조건식1, 조건식2가 모두 거짓일 경우 실행
	실행문1;
	실행문2;
}

사용
if문은 조건문의 참, 거짓 여부에 따라 실행 내용을 다르게 하고 싶을 때, 사용합니다. else if나 else 블록은 생략할 수 있습니다. if문, if ~ else문, if ~ else if ~ else 문 등 다양한 종류가 있습니다.

참고
if문을 이용해서 값의 범위에 따라 실행을 달리 하고 싶다면, 다음과 같이 범위가 가장 좁은 if문을 가장 먼저 오도록, 최상위에 위치하여야 합니다.
